# Neex Framework - Product Overview

## Purpose
Neex is a modern fullstack framework that combines Express.js backend capabilities with Next.js frontend power, providing developers with a unified development experience for building scalable web applications.

## Key Features

### Core Framework (`neex` package)
- **Parallel Command Execution**: Run multiple commands simultaneously with `neex p`
- **Development Server**: Integrated dev server with `neex dev`
- **Express + Next.js Integration**: Seamless backend and frontend development
- **TypeScript Support**: Full TypeScript integration out of the box

### Project Generator (`create-neex` CLI)
- **Interactive Project Setup**: Guided project creation with prompts
- **Multiple Package Managers**: Support for npm, pnpm, yarn, and bun
- **Template Generation**: Automated setup of backend, frontend, and configuration files
- **Prisma Integration**: Database schema and migration setup
- **Progress Tracking**: Real-time installation progress with retry mechanisms

### Documentation Site
- **Comprehensive Docs**: Complete framework documentation
- **MDX Support**: Rich content with React components
- **Search Functionality**: Easy navigation and content discovery

## Target Users

### Primary Users
- **Fullstack Developers**: Building modern web applications with TypeScript
- **Teams**: Requiring consistent project structure and development workflows

### Use Cases
- **Web Applications**: Full-stack applications with API and frontend
- **Rapid Prototyping**: Quick project setup and development
